# jemore-media-agency-simulation-game

Media Company Simulation Game

A 3D Unity-based simulation game designed as an experiential learning tool 
for undergraduate media and communication students.

## Built With
- Unity 6
- C#

## Play the Game
https://play.unity.com/en/games/14c170da-127b-449a-847a-c024f6626dfe/jemore-media-agency-the-internship

## About
Players take on the role of an intern at Jemore Media Agency and progress 
through four departments — Brand Management, Strategy, Art, and Production — 
completing tasks, mini-games, and quizzes before receiving a personalised 
career recommendation.
